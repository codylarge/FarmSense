### How to run it on your own machine

0. Create a venv
   ```
   $ python -m venv venv
   $ venv\Scripts\activate
   ```
   
1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
